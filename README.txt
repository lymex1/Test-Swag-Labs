Чтобы скачать все библиотеки нужно в терминал
прописать pip install -r requirements.txt

                        Чтобы запустить тесты надо:

            --------------------------1--------------------------
            Перейти в Recent Configurations (Слева от кнопки Run)
            --------------------------2--------------------------
                    Потом нажать на Egit Configurations
            --------------------------3--------------------------
            Нажать на плюс в правом верхнем углу и в Python tests
                            выбрать pytest
            --------------------------4--------------------------
                В разделе Target выбрать путь к папке проекта
            test_swag_labs, а в разделе Working directory указать
                          путь к папке tests
            --------------------------5----------------------------
            Осталось выбрать нашу конфигурацию и запускать проект ✅
            -------------------------------------------------------
